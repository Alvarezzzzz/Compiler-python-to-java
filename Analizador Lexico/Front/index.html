Bienvenido Lexer

El objetivo de este compilador es tomar código fuente Java como entrada y producir código fuente Python equivalente como salida. Esto implica analizar la estructura y el significado del código Java y luego generar código Python que reproduzca la misma funcionalidad.

Fases

  • Análisis Léxico (Lexer): Divide el código fuente en tokens.
  • Análisis Sintáctico (Parser): Construye el árbol de sintaxis abstracta (AST).
  • Análisis Semántico: Verifica la coherencia y las reglas del lenguaje.
  • Generación de Código Intermedio (Opcional): Transforma el AST en una representación intermedia.
  • Generación de Código Python: Genera el código Python equivalente.
